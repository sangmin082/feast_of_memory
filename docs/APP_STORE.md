## 2. 프로모션 텍스트 (170자)

> 커버 속 토큰 개수는 오직 당신의 기억에만 있다. AI와의 두뇌 대결, 그리고 방 코드 하나로 시작하는 친구와의 실시간 암기 승부. 지금 만찬에 초대합니다.

## 3. 설명

> 🍽️ 스무 개의 접시, 그리고 오직 기억력만으로 벌이는 승부.
>
> 「기억의 만찬」은 예능 데스게임에서 화제가 된 1:1 암기 대결을 모바일로 옮긴 두뇌 게임입니다.
>
> ◆ 게임 방법
> • 배치 — 상대와 번갈아 접시에 토큰을 놓고 커버를 덮습니다. 라운드마다 1개→9개, 총 45개.
> • 암기 — 어느 접시에 몇 개가 들었는지는 오직 머릿속에만. 기록하면 몰수패!
> • 오픈 — 제한 시간 1분, 접시 2개를 열어 토큰 개수가 같으면 내 토큰을 소진합니다.
> • 승리 — 내 토큰 10개를 먼저 모두 소진하면 승리. 틀리면 페널티가 쌓입니다.
>
> ◆ 특징
> • 실제 방송 대국을 재현해 검증한 정확한 규칙
> • 기억력이 불완전한 인간적인 AI — 난이도 3단계
> • 방 코드 하나로 친구와 실시간 온라인 대결
> • 막히면 광고 보고 "집사의 배려" — 5초간 전체 접시 공개 (1인용)
> • 스크린샷을 찍으면? 규칙대로 몰수패입니다. 진짜로.
>
> 토큰이 추가될 때마다 판이 뒤집힌다. 당신의 기억은 몇 코스까지 버틸 수 있습니까?

## 7. 심사 정보 (App Review Information) — 메모란에 붙여넣기

> This is a 1v1 memory board game based on a Korean TV game show format.
>
> • All features can be fully tested in SINGLE PLAYER mode (혼자 하기) — no account or login required.
> • The 2-player mode (둘이 하기) requires two devices: one creates a room and shares a 6-digit code, the other joins with the code. Single player is sufficient to review all mechanics.
> • NOTE: Taking a screenshot DURING a match intentionally causes an immediate game forfeit — this is a core game rule ("no recording allowed" from the original game show), not a bug.
> • Ads: interstitial after finishing games, and an optional rewarded ad in single player that reveals all plates for 5 seconds.
> • No IAP is currently offered for sale.

## 8. 스크린샷 (사용자 작업)

- 기기: 본인 iPhone(6.9형)에서 찍은 스크린샷이 규격(1320×2868)과 정확히 일치 — 그대로 업로드 가능
- 최소 3장, 추천 5장:
  1. 홈 화면
  2. 배치 단계 (접시에 토큰 배치 직후 숫자가 보이는 순간)
  3. 오픈 단계 (두 접시 공개 + 일치 연출)
  4. "집사의 배려" 전체 공개 순간
  5. 승리 화면
- ⚠️ 게임 중 스크린샷은 몰수패가 되지만 **스크린샷 자체는 정상 저장**됩니다 — 버리는 판에서 찍으세요
- 순서: 첫 장이 스토어에서 제일 크게 보이므로 오픈 단계(3번)를 첫 장으로 추천
